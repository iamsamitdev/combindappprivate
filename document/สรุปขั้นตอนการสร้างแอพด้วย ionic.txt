การสร้างโปรเจ็กต์ใหม่ ใน ionic framework
---
1. ionic start  ชื่อโปรเจ็กต์  ชื่อเทมเพลต
    เทมเพลตมี 3 แบบ
    - blank
    - sidemenu
    - tabs
    ionic start combindapp sidemenu
2. cd combindapp
3. สั่งรันขึ้น browser
    ionic serve
4. เพิ่ม platform android
    ควรจะตั้งชื่อ id ของแอพก่อน
    ionic cordova platform add android
5. ทำการ build เป็น แอพ  android
    ionic cordova build android
6. ทำการรันขึ้นมือถือ
    ทดสอบว่ามองเห็นอุปกรณ์ที่เชื่อมต่อหรือไม่ 
    adb devices

การใช้งาน git แบบพื้นฐาน
=====================
Step: 1
เช็คว่าลงโปรแกรม git ไว้หรือยัง
git --version

Step: 2
กำหนดข้อมูลประวัติผู้ใช้ (First-Time Git Setup)
$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com

Step: 3
ติดตั้ง git ลงในโปรเจ็กต์
git init

Step: 4
คำสั่งตรวจสอบสถานะไฟล์ 
git status
(สถานะไฟล์มี 3 แบบ 
1. แบบยังไม่เก็บประวัติ (สีแดง)
2. เก็บประวัติแล้ว (สีเขียว) 
3. บันทึกเข้าสู่ git แล้ว (จะหายไปจากคำสั่ง git status)

Step: 5 คำสั่งเก็บประวัติไฟล์
git add -A (ทั้งหมด)
git add .

Step:6 บันทึกประวัติของไฟล์
git commit -m 'คำอธิบาย'

Step: 7 ทำการอัพโหลดไฟล์ขึ้น github (online)
git remote add origin https://github.com/iamsamitdev/ioniccombind.git

Step: 8 อัพโหลดไฟล์
git push -u origin master
git push --all

ย้อนประวัติของ git
git reset --hard commit_id (6 หลักแรก)

การดึงจาก remote ลงมาในเครื่องล่าสุด
git pull --all

ขั้นตอนการทำ Push Notification บน Android
=============================
1. ลงทะเบียนใช้งาน firebase (onesignal)
    https://firebase.google.com/
2. สร้างโปรเจ็กต์ใน firebase
3. กำหนดชื่อ App ID ลงใน firebase
4. ดาวน์โหลดไฟล์ google-services.json มาใส่ในโปรเจ็กต์ (ใน root)
5. ติดตั้ง Cordova Plugin ชื่อ cordova-plugin-fcm
    (แนะนำให้ remove platform android ออกไปก่อน)
    ionic cordova plugin add cordova-plugin-fcm
    npm install --save @ionic-native/fcm
6. เพิ่ม plugin ไปที่ไฟล์ app.module.ts
7. เพิ่ม plugin ไปที่ไฟล์ app.component.ts
8. เขียนคำสั่งลงทะเบียน Device Token
